How it works

All-in-one: Open any DIPI page → tap bookmark
→ Three buttons appear bottom-right: 🔍 Audit · 📥 Scrape · 📷 Photos
→ Tap the tool you need — it loads fresh from this site

Scraper-only flow:
Tap 1: Open any DIPI page → tap bookmark
→ Shows upcoming courses for Dhamma Sudha
→ Pick a course → auto-navigates to search page

Tap 2: Tap bookmark again on the search page
→ Auto-scrapes all applicants + phone numbers
→ Tap "Open Tracker" → app loads with all data

Vipassana Call Tracker

The call tracker now runs directly on the dipi applicants page — no separate web app to open, and your call statuses stay on the page you scraped from.

How to use it
  1. Install the Chrome extension, Tampermonkey userscript, or bookmarklet (see the repo).
  2. Open a course on dipi.vridhamma.org and click 🔄 Scrape.
  3. Click Open Inline Call Tracker — the calling dashboard opens right there.
